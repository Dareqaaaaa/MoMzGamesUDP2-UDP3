ไฟล์ button.txt:
ใส่แค่ ตัวเลขเท่านั้น ตัวเลขที่ใส่ได้ 1, 2, 3, 4
1 คือ เปิดปุ่ม Check เปิดปุ่ม Start
2 คือ ปิดปุ่มทั้งหมด
3 คือ ปิดปุ่ม Check เปิดปุ่ม Start
4 คือ ปิดปุ่ม Check เปิดปุ่ม Start แต่จะไม่สามารถกดเข้าเกมได้ เพราะ ปิดปรับปรุงเซิร์ฟจาก File status.txt

ไฟล์ check.txt:
ใส่ตัวอักษรได้ทั้งหมด จะพิมอะไรก็ได้ เช่นยกตัวอย่าง: ระบบ Check File ตัวเกมยังไม่สมบูรณ์!!!
จะแจ้งเตือนตอนกดปุ่ม Check ใน โปรแกรม

ไฟล์ status.txt:
สถานะ PBLauncher.exe ตอนเปิดโปรแกรม
ใส่ได้แค่ ตัวเลขเท่านั้น ตัวเลขที่ใส่ได้ 0,1,2,3
0 คือ ปิดเข้าเกม
1 คือ เปิดปกติ
2 คือ ปิดปรับปรุงเซิร์ฟเวอร์
3 คือ ตัดการเชื่อมต่อ

ไฟล์ text.txt:
ใส่ตัวอักษรได้ทั้งหมด จะพิมอะไรก็ได้ เช่นยกตัวอย่าง: ปิดปรับปรุงเซิร์ฟเวอร์!!! สาเหตุ: แก้ไขระบบ
จะแจ้งเตือน ตอนเข้าโปรแกรมถ้าในไฟล์ button.txt ใส่ เลข 4 และ ถ้าใส่เลขในไฟล์ status.txt เป็น
เลข 2 จะแจ้งตอน เปิดโปรแกรมแรกๆ
